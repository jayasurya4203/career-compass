# Career Compass

AI-powered career recommendation, resume analysis, and employability enhancement platform (Flask + React + scikit-learn).

## Quick start

### 1. Backend (Python 3.10+)

```powershell
cd career-compass\backend
python -m pip install -r requirements.txt
python scripts\generate_dataset.py
python scripts\train_career_model.py
python app.py
```

API runs at `http://127.0.0.1:5000`. SQLite database and uploaded resumes are stored under `backend/`.

### 2. Frontend (Node 18+)

```powershell
cd career-compass\frontend
npm install
npm run dev
```

Open `http://127.0.0.1:5173`. The dev server proxies `/api` to Flask.

### MySQL (optional)

Set `DATABASE_URL` to a SQLAlchemy URI (for example `mysql+pymysql://user:pass@localhost/career_compass`) and install a MySQL driver such as `pymysql`. The default is SQLite for easy college demos.

### Admin analytics (optional)

`GET /api/admin/summary` with header `X-Admin-Key: demo-admin` (override with env `ADMIN_KEY`).

## Project structure

- `backend/app.py` — REST API (auth, profile, quiz, resume, ML, ATS, progress, chatbot)
- `backend/career_engine.py` — hybrid career scoring, skill gap, roadmap, TF–IDF matching, ATS
- `backend/data/` — synthetic training CSV, role catalog, course/project/interview JSON
- `backend/ml_artifacts/career_rf.joblib` — trained RandomForest pipeline (generated by training script)
- `frontend/src` — React Router UI (Tailwind + Recharts)
